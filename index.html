Beat Revival





Welcome to the Beat Revival!

This project aims to bring back the online functionality of Mirror's Edge: Catalyst through the reverse engineering of EA/DICE's server software.

This project is still in development and isn't ready just yet but if you would like to keep up-to-date with the project's development then you can join our Discord (click the Discord tab at the top).


If you would like to find out more about the project, click the "Info" tab.
